# vue3-yanxuan

这是一个基于 `Vue 3 + Vue Router 4 + Vuex 4 + Axios + Less + Element Plus` 的网易严选前端练习项目。

## 当前项目重点

当前代码重点不是业务做完多少，而是已经开始形成比较清晰的前端工程结构，包括：

- 入口统一装配
- 路由与 Layout 布局
- Vuex 模块化
- Axios 二次封装
- 开发代理
- 头部分类数据流
- 首页轮播模块
- 首页新鲜好物模块
- 首页人气推荐模块
- 首页产品区块模块
 - 首页骨架屏占位组件
- 全局组件注册
 - 加载态组件拆分
- 通用面板组件复用
- 商品卡片组件复用

## 当前进度

已完成：

- 项目基础骨架
- 顶部通栏、头部、分类导航
- 分类接口请求与渲染
- 首页轮播图模块
- 首页新鲜好物模块
- 首页人气推荐模块
- 首页产品区块模块
 - 首页产品区块模块
 - `HomeNewSkeleton` / `AppSkeleton` 骨架屏组件
- `AppBanner`、`AppMore` 全局组件
- `MyPanel` 通用结构组件
- `MyGoodsItem` 商品卡片组件

未完成：

- 分类页真实内容
- 登录完整业务
- 商品详情、购物车、订单流程
- 首页更多内容模块

## 运行方式

```bash
npm install
npm run serve
```

打包：

```bash
npm run build
```
